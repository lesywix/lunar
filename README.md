# lunar

`lunar` 是一个命令行下的公历农历查询转换工具。

## 安装
### Homebrew
```
brew tap xwjdsh/tap
brew install xwjdsh/tap/lunar
```
### Go
```
go get -u github.com/xwjdsh/lunar/cmd/lunar
```
### Manual
从 [releases](https://github.com/xwjdsh/lunar/releases) 下载可执行文件并将其放到 PATH 环境变量对应的路径中。

## 使用
```
> lunar -h
NAME:
   lunar - lunar is a command line tool for conversion between Gregorian calendar and lunar calendar.(1901~2100)

USAGE:
   lunar [global options] command [command options] [arguments...]

COMMANDS:
   alias, a        show alias date info
   reverse, r      reverse mode, query date by lunar date
   solar-term, st  get solar term info by name
   help, h         Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --format value, -f value  output date format (default: "2006-01-02")
   --year value, -y value    target year (default: 2021)
   --help, -h                show help (default: false)

```

### 公历农历转换
```
> # lunar -y 2022 # 指定年份，月日为今日
> # lunar -y 2022 0701 指定年月日
> lunar # 不带参数年月日为公历今日
+------------+------------+--------+------+------+------+
|    公历     |    农历     |  星期  | 节气  | 别名  | 标签 |
+------------+------------+--------+------+------+------+
| 2021-08-04 | 2021-06-26  | 星期三 |      |      |      |
+------------+------------+--------+------+------+------+

> # lunar -r -y 2022 # 查询农历，指定年份
> # lunar -r -y 2022 0701 # 查询农历，指定年月日
> lunar -r # 查询农历，不带参数年月日为公历今日
+------------+------------+--------+------+------+------+
|    公历     |    农历     |  星期  | 节气  | 别名  | 标签 |
+------------+------------+--------+------+------+------+
| 2021-09-10 | 2021-08-04 | 星期五  |      |      |      |
+------------+------------+--------+------+------+------+
```
